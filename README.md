# Orça Fácil - Página de Downloads

Esta é a página de apresentação e de transferência para o **Orça Fácil**, a aplicação definitiva para profissionais independentes gerarem orçamentos profissionais, recibos e gerirem os seus clientes. 

## 🚀 Funcionalidades da Aplicação

O Orça Fácil foi desenhado para simplificar o dia a dia de trabalhadores autónomos e freelancers, oferecendo as seguintes ferramentas:

* **Geração de PDFs:** Crie orçamentos e recibos em PDF com uma aparência profissional em questão de segundos, prontos para enviar via WhatsApp ou E-mail.
* **Gestão de Clientes:** Mantenha um catálogo devidamente organizado com todos os dados dos seus clientes e o histórico de serviços prestados.
* **Dashboard Financeiro:** Acompanhe os seus lucros, visualize métricas importantes e tenha total controlo sobre a faturação do seu negócio.
